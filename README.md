## 📖 项目简介

> **LinuxDo Scripts** 是一个功能丰富的浏览器扩展，专为提升 LinuxDo 论坛的使用体验而设计。它集成了从基础界面优化到高级 AI 辅助的多项实用功能，让您的论坛浏览和互动体验更加流畅高效。

📢 **论坛交流贴**：[linuxdo 增强插件，持续更新欢迎反馈（浏览器扩展，油猴已失效不再更新）- LINUX DO](https://linux.do/t/topic/170951)

### ✨ 核心亮点
- 🎨 **界面美化** - 多种主题皮肤，个性化定制
- 🤖 **AI 智能** - 话题总结、智能回复生成
- 📚 **内容管理** - 收藏夹、用户标签、内容过滤
- ⚡ **体验优化** - 快捷操作、自动化功能
- 🔧 **高度可定制** - 支持自定义 CSS、快捷回复等

## 🛠️ 开发指南

### 📋 环境要求
- **Node.js**: `v22.12.0`
- **包管理器**: `npm`

### 🚀 本地开发

```bash
# 1. 克隆项目
git clone https://github.com/ezyshu/linuxdo-scripts.git
cd linuxdo-scripts

# 2. 安装依赖
npm install

# 3. 开发模式
npm run dev              # Chrome 开发模式
npm run dev:firefox      # Firefox 开发模式
```

### 📦 加载扩展

<table>
<tr>
<td width="50%">

**Chrome/Edge/Brave**
1. 打开 `chrome://extensions/`
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `.output/chrome-mv3` 文件夹

</td>
<td width="50%">

**Firefox**
1. 打开 `about:addons`
2. 点击设置图标
3. 选择"调试附加组件"
4. 点击"临时载入附加组件"
5. 选择 `.output/firefox-mv2` 文件夹中的 `manifest.json`

</td>
</tr>
</table>


### 🏗️ 项目架构
```
linuxdo-scripts/
├── 📁 entrypoints/          # 扩展入口文件
│   ├── background.ts        # 后台脚本
│   ├── content.ts          # 内容脚本
│   └── popup/              # 弹窗页面
├── 📁 components/           # Vue 组件
│   ├── common/             # 通用组件
│   └── features/           # 功能组件
├── 📁 public/              # 静态资源
│   ├── icon/               # 图标文件
│   └── images/             # 图片资源
├── 📁 docs/                # 项目文档
├── 📁 i18n/                # 国际化文件
├── 📁 plugin/              # 插件配置
├── 📄 wxt.config.ts        # WXT 配置文件
├── 📄 package.json         # 项目配置
└── 📄 tsconfig.json        # TypeScript 配置
```
